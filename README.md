### About

I'm a **16-year-old developer** from Heide, Schleswig-Holstein. I build AI systems that actually matter.

My main project is [**SterilVision**](https://sterilvision.de) — a Computer Vision system that detects defects in surgical instruments before they reach the OR, built in collaboration with WKK Heide hospital.

```text
Current focus  Computer Vision · Deep Learning on Edge Hardware
Hardware       Raspberry Pi 5 → 80 ms inference · 20 W
Stack          Python · PyTorch · OpenCV · TypeScript · React
Location       Heide, Schleswig-Holstein 🇩🇪
```

- 🔬 **BWKI 2026** — AI for sterile instrument inspection
- 🏥 **WKK Heide** — deployed and tested at a real hospital AEMP lab
- ✏️ Building **[pixlang](https://github.com/hackbert301009/pixlang)** — my own programming language
- 🌍 Standing for humanity, open knowledge, and tech that has a point
